Analyze the Spec

~30 seconds

Your Goal

Understand what the business is asking for and how it maps to your technical systems.

Ask Bob:

Please analyze the SSO_MFA_Integration.pdf spec and tell me:
1. What systems are involved?
2. What are the current security gaps?
3. What are the SOC 2 compliance requirements?
4. What technology stacks are we working with?

What Bob Returns

Systems Overview

3 repositories identified:

  • auth-service (Node.js/Express) - OAuth token issuance
  • user-service (Java/Spring Boot) - User management, authentication
  • frontend (React) - Customer portal UI

Security Vulnerabilities Found

  • Hardcoded JWT secret in tokenController.js
  • CSRF protection disabled in SecurityConfig.java
  • Authentication tokens stored in localStorage (XSS vulnerable)
  • No MFA enforcement (SOC 2 CC6.1 requirement)
  • No session timeout (SOC 2 CC6.6 requirement)

SOC 2 Compliance Requirements

  • CC6.1-2026-01: Implement MFA for all users
  • CC6.6-2026-03: Enforce 15-minute session timeout
  • CC7.2-2026-04: Add authentication event logging
  • CC8.1-2026-05: Remove hardcoded secrets
